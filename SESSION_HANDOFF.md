### Text Rendering Quality
The AI model (Gemini 3 Pro) struggles with:
- Consistent text styling across pages
- Dialogue box placement (some float instead of being on screens)
- Text legibility in some panels
- Maintaining exact style specifications

**Pages with notable text issues:**
- **Page 6:** Bottom caption has mixed styling
- **Page 7:** Purple background bleeding, some Woo dialogue placement
- **Page 8:** Some text boxes floating instead of on laptop screen
- **Page 11:** Occasional floating purple boxes

### Style Consistency Target (Not Fully Achieved)
**Intended style guide:**
- Narrative captions: Tan/yellow rectangular boxes (like Page 2 top left)
- Hero dialogue: White speech/thought bubbles (like Page 1)
- Woo app: Purple text on phone/laptop screens only (like Page 2 phone)

**Reality:** About 70% consistent. Some pages have purple dialogue boxes floating in space instead of on screens.
