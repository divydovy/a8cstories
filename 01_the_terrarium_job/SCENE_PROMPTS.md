# The Terrarium Job - Detailed Scene Prompts for Image Generation

## Style Guide for All Scenes

**Global Art Direction:**
- Style: Graphic novel illustration, neo-noir tech aesthetic, clean vector-style art
- Rendering: Semi-realistic digital illustration with strong shadows and highlights
- Color grading: Cinematic color correction, teal and orange color theory
- Line work: Clean, confident lines with strategic line weight variation
- Lighting: Dramatic but naturalistic, strong key lighting
- Composition: Rule of thirds, dynamic angles, clear focal points

**Consistency Notes:**
- Protagonist: 30-something, gender-neutral, casual tech aesthetic, short dark hair, comfortable clothing (hoodies, simple t-shirts)
- Terrarium design: Geometric cubic frames, modular snap-together design, glossy black PLA material
- Digital elements: Abstract flowing data, geometric UI patterns, glowing teal/blue holograms
- NO READABLE TEXT in any scene (use abstract UI elements only)

## Generation Notes

### Prompt Strategy
- Each prompt is detailed enough to generate consistent results
- Style keywords repeated for consistency: "graphic novel", "neo-noir", "clean vector illustration"
- Lighting and color palette explicitly described
- Character consistency maintained through repeated description
- Camera angles specified for cinematic quality
- Mood explicitly stated to guide generation
- **NO READABLE TEXT** - all digital elements are abstract shapes, icons, and visualizations

### Post-Generation Process
1. Generate each scene
2. Review for consistency
3. Refine prompts if needed and regenerate
4. Ensure visual continuity across all 12 scenes
5. Compile into final graphic novel sequence

### Variations to Consider
- May need to generate multiple versions of each scene to get optimal results
- Can adjust lighting, composition, or details based on initial results
- Character consistency is priority - may need reference images
